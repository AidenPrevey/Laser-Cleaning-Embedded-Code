Constructs a \doxylink{class_serial_receiver_transmitter_1_1_command_message}{Command\+Message} object by parsing a buffer containing G-\/code or M-\/code commands. 

This constructor takes a character buffer as input, which represents a G-\/code or M-\/code command string. It parses the buffer to identify the command type (G or M) and its associated parameters. The parsed data is stored in the respective command fields (e.\+g., G0, G4, M80, etc.) of the \doxylink{class_serial_receiver_transmitter_1_1_command_message}{Command\+Message} object.

The parsing logic handles\+:
\begin{DoxyItemize}
\item G-\/code commands (e.\+g., G0, G4, G28, G90) and their parameters (e.\+g., Y, A, C).
\item M-\/code commands (e.\+g., M80, M17, M906) and their parameters.
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em buffer} & A null-\/terminated character array containing the G-\/code or M-\/code command string.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The constructor uses {\ttfamily strtok} to tokenize the input buffer and processes each token to extract the command type and parameters. Unhandled commands or parameters are logged using {\ttfamily Serial.\+print}. 
\end{DoxyNote}
